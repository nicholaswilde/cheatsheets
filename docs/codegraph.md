# codegraph

Codegraph - Provides source symbol search and semantic code traversal using a local codebase index.

Search and explore source code symbols semantic structure:
Call codegraph_explore with: query, projectPath, maxFiles
Example search for AuthService initialization:
Call codegraph_explore with: query="AuthService loginUser session-manager"
