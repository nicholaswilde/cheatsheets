# bun

All-in-one JavaScript runtime, packager, bundler, and test runner (https://bun.sh)

!!! info "To install bun"
    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```

!!! info "To initialize a new project"
    ```bash
    bun init
    ```

!!! info "To run a file or package script"
    ```bash
    bun run <file_name>
    ```

!!! info "Or"
    ```bash
    bun run <script_name>
    ```

!!! info "To install dependencies"
    ```bash
    bun install
    ```

!!! info "To add a package to dependencies"
    ```bash
    bun add <package_name>
    ```

!!! info "Or as a devDependency"
    ```bash
    bun add -d <package_name>
    ```

!!! info "To remove a package"
    ```bash
    bun remove <package_name>
    ```

!!! info "To run a binary or package without installing (like npx)"
    ```bash
    bunx <command>
    ```

!!! info "To run tests"
    ```bash
    bun test
    ```

!!! info "To run a script in hot-reload mode"
    ```bash
    bun --hot <file_name>
    ```
