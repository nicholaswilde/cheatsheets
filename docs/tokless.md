# tokless

CLI tool designed to simplify the installation and management of token-saving plugins for AI coding assistants (https://github.com/HoangP8/tokless)

!!! info "To install and configure tokless plugins (e.g., rtk, caveman, codegraph)"
    ```bash
    tokless
    ```

!!! info "To display the current configuration status and verify installed tools"
    ```bash
    tokless doctor
    ```

!!! info "To build a project code graph index in the current directory"
    ```bash
    tokless index
    ```

!!! info "To check for updates and upgrade installed plugins"
    ```bash
    tokless update
    ```

!!! info "To uninstall tokless and remove all changes made to the system"
    ```bash
    tokless uninstall
    ```
